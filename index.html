Featured Projects

MusicPal

The final project for DigitalCrafts, MusicPal is a web app that connects users who have similar taste in music and are looking for a pal to attend live music events with. Users can build a profile with shows they are interested in, as well as their music tastes and likes and dislikes. A single event will show other members who are interested so that connections can be made. Built with JavaScript, React.js, Node, Express.js, PostGreSQL, Eventful API, and Lastfm API.

PhotoSpotter

Built with: HTML5, CSS3, Javascript, Google Maps API, Flickr API, OpenWeatherMap API

PhotoSpotter is an interactive Web App that gives users the power to recreate their favorite pictures by giving them the location, time and camera information of the image, as well as local weather. The user can input the subject they would like to photograph and instantly be transported to a gallery of their specified interest. With one click of an image, the user will immediately receive all the necessary information to create a similar photo.

Lender-Be

Built with: PostgreSQL , Node.js, Express, HTML5, CSS3, JavaScript, Bcrypt

A "social lending" site. Users can list any items they are willing to lend. Friends, family, and neighbors can connect and search to see if someone has something they need, from a book to a power tool. People can request to borrow and, if accepted, there is a record of who has the item, a field where the borrower puts how long they need it, another where the lender can put when they need it back, and the app can send reminders of the "due date."
